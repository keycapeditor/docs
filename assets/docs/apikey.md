# Add your own API key

1. Head to [Cerebras AI](https://www.cerebras.ai/)
2. Create a account and sign in
3. Get to the API tab
4. Copy your API key
5. Into the Keycap Editor click the **API key**
6. Add your API key there and click save

Now you can get more AI responses
